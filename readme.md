# FSD practice 
## Task 2 TOXIC

### Установка

_Клонировать_
> ```git clone https://github.com/muromtsev/toxic.git```

_Установка_
> ```npm i```

_Запуск webpack server_
> ```npm run dev```

_Запуск production build_
> ```npm run build```
